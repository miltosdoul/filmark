## Use locally

With [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) installed on your computer, run:

```bash
# Clone repository
$ git clone https://github.com/miltosdoul/filmark

# Go into the repository
$ cd filmark

# Install dependencies
$ npm install

# Run the website
$ npm start
```
